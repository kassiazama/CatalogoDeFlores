# Catálogo de flores 🌸
Projeto em Java com Spring Boot desenvolvido como requisito do Bootcamp Java da Ti Academy. <br>

- Spring Data JPA; <br>
- Lombok 🌶; <br>
- Versionamento de banco com Liquibase; <br>
- Banco de dados em MySQL. 🐬
